# Next.js Sales Dashboard 📊

A basic sales dashboard application built using **Next.js 15**, **TypeScript**, and **Tailwind CSS**, following the **Atomic Design Principle**.  
The project visualizes yearly sales data (2022, 2023, 2024) using charts and mock data.

---

## 🚀 Project Overview

This project demonstrates how to build a modular and scalable frontend application using modern web technologies.  
It displays sales data for multiple years using chart components and a clean dashboard layout.

Key focus areas:
- Atomic component structure
- Chart-based data visualization
- Clean UI with reusable components
- Scalable architecture for future enhancements

---

## 🧱 Tech Stack

- **Next.js 15**
- **TypeScript**
- **Tailwind CSS**
- **Recharts** (for charts)
- **Mock sales data** (inspired by Kaggle datasets)

---

## 🧩 Project Structure (Atomic Design)

The project follows the **Atomic Design Principle**:

- **Atoms** – Basic UI elements (buttons, labels, inputs)
- **Molecules** – Small combinations of atoms (chart cards, filters)
- **Organisms** – Complex UI sections (sales charts)
- **Pages** – Dashboard page integrating all components

This structure makes the codebase easy to maintain and scale.

---

## 📊 Features

- 📈 Sales visualization for **2022, 2023, and 2024**
- 📊 Multiple chart components
- 🧩 Reusable and modular components
- 🖥️ Empty dashboard page layout
- 🎨 Responsive UI using Tailwind CSS

---

## 🛠️ Implementation Details

- Created a **dashboard page** using Next.js App Router
- Built **separate chart components** using Recharts
- Used **mock sales data** to simulate real-world datasets
- Integrated components into the dashboard page
- Ensured clean folder structure and reusable logic
